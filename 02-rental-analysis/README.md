# Rental Analysis

This section contains SQL examples focused on rental activity and rental-related reporting using the Sakila database.

## Included examples

### 1. Payments and Rentals
A basic SQL join between the `payment` and `rental` tables showing payment and rental details such as payment ID, rental ID, amount, rental date, and payment date.

### 2. Rental Report
A more advanced rental report combining rental, payment, inventory, and film data. The output includes rental details, film information, payment information, and clearly named business-friendly columns.

## Skills demonstrated
- INNER JOIN
- multi-table joins
- aliasing
- reporting-oriented column naming
- combining transaction and descriptive data

## Dataset
Sakila sample database
