# SQL Portfolio

Portfolio of SQL projects focused on customer, payment, revenue, and rental analytics using the Sakila database.

## About this repository
This repository contains business-oriented SQL projects that demonstrate practical analytical skills such as data extraction, joins, aggregations, views, validation queries, and reporting logic.

## Projects
- Customer Payment Report
- Rental Report
- Unpaid Rentals Analysis
- Revenue by Film
- Most Profitable Film
- Advanced SQL: Ranking, Cumulative Sum, Pareto Analysis

## SQL skills demonstrated
- SELECT, WHERE, ORDER BY
- GROUP BY, HAVING
- JOINs
- Subqueries
- Views
- Window functions
- Date functions
- Validation queries

## Dataset
Most examples are based on the Sakila sample database.

## Goal
The goal of this portfolio is to present SQL solutions in a business and reporting context, not just isolated technical exercises.
